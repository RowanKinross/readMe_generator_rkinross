# Sample README

## Project Status
This project is on pause, for now.

## Description
Sample README is a document that demonstrates what a sample readme would look like if made using the readme generator. The aim of the project was to generate a readme which could be used for submission in this week's assignment. Users can read this readme and determine if the generator is proficient. The project makes use of Javascript, Node.js, inquiry (all through the readme generator). <br>
